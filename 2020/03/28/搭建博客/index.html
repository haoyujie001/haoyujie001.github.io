搭建一个hexo+github博客的方法

发表于 2020-03-28 热度： Valine：
本文字数： 1.3k 阅读时长 ≈ 7 分钟

2020/3/27，自己的Hexo博客总算run起来了，算上优化主题，来来回回花费了三天时间才弄好，下面来分享一下搭建一个hexo博客的方法

01.安装 Node.js

=====================================

打开官方网站
https://nodejs.org


安装步骤非常简单，一直next，下一步就可以了，默认安装就行。

02.安装 Git

===================================

打开官方网站
https://git-scm.com/downloads
然后我们选择windows版本的下载

安装也是一直点下一步，安装官方默认的来就行，
完了，在开始菜单可以看到

安装完Git别忘了配置环境变量！

tips：这个Git Bash下载下来就相当于Linux中的终端窗口了，以后我们就用这个东西来打开终端。

03.安装hexo

===================================

先在终端输入node -v 和 npm -v（若出现版本号则安装成功）

看看 ==node，npm== 是否安装成功，没有成功的就重新安装node。

我们需要先来安装个cnpm提高速度，以后下载什么东西都用cnpm

在上面终端继续输入
npm install -g cnpm --registry=https://registry.npm.taobao.org
下载完后继续输入 cnpm -v验证是否安装成功

==完成之后安装hexo==
cnpm install -g hexo-cli

==验证是否安装成功==
hexo -v
出现,说明成功

我们可以在想要保存博客的地方创建文件夹（mkdir blog)

然后我们运行下面命令，这步是关键，主要是建立整个项目。
hexo init
之后打开blog目录如下：

04.运行测试

=============================

这里我们经常用到的有三个命令

1.hexo clean #用来清理缓存文件
2.hexo g #部署文件
3 hexo s #运行本地服务器
4 hexo d #上传到服务器

我们运行
hexo s

打开浏览器，输入localhost:4000,看到

说明本地环境搭建已完成

05.把博客部署到github上

=====================================
首先我们登录github官网，注册，登录，新建一个数据库，数据库名称必须和自己账号昵称一样：xxx.github.io
这个数据库名字就是以后我们访问博客的链接。

06.配置_config.yml 文件

=====================================
打开该文件，翻到最后
在最后加上下面三个属性
==type: ‘git’
repo: https://github.com/haoyujie001/haoyujie001.github.io.git
branch: master==
注意每个冒号后面有一个空格

07.上传到服务器

=====================================
终端输入hexo d
然后输入你的github账号密码
会提示推送成功
==输入时密码是不会显示的哦==
然后刷新你的GitHub数据库会发现多了很多文件
之后我们就可以复制仓库名字来访问我们的博客了
